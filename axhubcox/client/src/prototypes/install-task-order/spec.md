# 安装任务报单原型规格

## 页面定位

该原型用于展示安装任务报单流程，面向安装人员或业务人员录入车辆、安装人员、拆装套餐、主机品牌、产品选型和报单图片等信息。

## 页面结构

- 顶部页签：基本信息、报单信息、报单产品、报单图片。
- 基本信息区：维护车牌号、车架号、车型、安装人员、地址、备注等字段。
- 报单信息区：维护旧主机编码、拆装套餐、主机品牌、地区等业务信息。
- 报单产品区：展示产品槽位、组合商品选择、设备选择与扫码状态。
- 报单图片区：上传和展示报单相关图片。
- 底部操作区：保留当前提交与流程操作。

## 交互说明

- 点击页签切换不同报单信息区域。
- 点击车牌号进入车牌输入面板。
- 点击产品槽位进入设备选择页面。
- 组合商品选择会按现有逻辑填充可选产品槽位。
- 图片上传、地区/人员/品牌选择器保持当前原型行为。

## Axure 导出说明

- 入口文件：`index.tsx`。
- 导出组件名：`Component`。
- 当前不额外接入 Axure API，仅满足 Axure 导出静态检查要求。
