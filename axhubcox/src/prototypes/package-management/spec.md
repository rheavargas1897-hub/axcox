# 销售订单下单原型规格

## 页面定位

该原型用于展示销售订单下单与订单列表管理流程，入口为销售订单列表页，保留 v0 转换后的现有业务行为、交互和视觉表现。

## 页面结构

- 订单列表：展示销售订单相关记录与主要操作入口。
- 新建订单：通过现有页面交互进入下单流程，维护客户、销售、产品、质保、安装服务等信息。
- 订单详情：展示订单基础信息、商品信息、费用信息及相关业务状态。

## 交互说明

- 用户从订单列表查看订单数据。
- 用户通过列表中的操作入口进入新建或详情流程。
- 页面内的抽屉、表单、选择器、金额汇总等交互保持原实现不变。

## Axure 导出说明

- 入口文件：`index.tsx`。
- 导出组件名：`Component`。
- 当前不额外接入 Axure API，仅满足 Axure 导出静态检查要求。
